A API scaffold built with express.

## Commands


Install dependencies:

```bash
$ yarn install
```

Start app:

```bash
$ yarn start
```

Run on development:

```bash
$ yarn start:dev
```

Run on Docker:

```bash
$ bash docker.sh start
```

Run tests:

```bash
$ yarn test
```

Run linter:

```bash
$ yarn lint
```

Create new modules:

```bash
//generates controller and service
$ yarn generate module <name>

$ yarn generate controller <name>

$ yarn generate service <name>
```
